# Usability Testing Playbook

## Cadence
- Run one moderated usability round every month.
- Recruit 5 participants per round: 3 English, 2 Dutch.
- Test on desktop and mobile (at least 2 mobile sessions).

## Core Journeys
1. Landing page to login request.
2. Verify login code and reach dashboard.
3. Start a fit session and complete questionnaire.
4. Reach results and send report by email.

## What To Capture
- Time to complete each journey step.
- Where users hesitate, backtrack, or ask for clarification.
- Errors encountered (login send/verify, questionnaire completion, report send).
- Navigation confusion points on dashboard and mobile menus.

## Monthly Review Output
- Top 3 friction points ranked by impact and frequency.
- Proposed fixes with owner and target release.
- Success metric trend against prior month:
  - Landing -> start fit conversion
  - Mobile drop-off in dashboard/fit pages
  - Questionnaire completion rate
  - Verification-code completion rate
  - Support tickets related to login/navigation
